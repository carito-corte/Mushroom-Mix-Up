# 🍄 Mushroom Mix-Up

## Descripción

**Mushroom Mix-Up** es un videojuego desarrollado con **Unity** como parte del curso **Programación para Videojuegos** de la Universidad Nacional Abierta y a Distancia (UNAD). 
El proyecto se realizó de manera colaborativa utilizando **GitHub** como herramienta de control de versiones, lo que permitió la integración de los aportes de cada integrante del equipo.

El objetivo del juego es ofrecer una experiencia interactiva en la que el jugador deberá superar los desafíos propuestos mediante la exploración del escenario, la interacción con diferentes elementos y el cumplimiento de los objetivos planteados durante la partida.

# Mecánicas del juego

1. Movimiento del personaje.
2. Interacción de objetos del escenario.
3. Sistema de puntuación.
4. Condiciones de victoria y derrota.
5. Música ambiental y efectos de sonido.

# Controles

| Acción | Tecla |
|---------|-------|
| Moverse | W, A, S, D o Flechas |
| Saltar | Espacio |
| Aplastar | Espacio + Ubicar sobre NPC |

# Tecnologías utilizadas

GitHub
GitHub Desktop

# Instalación

1. Clonar el repositorio.

```bash
git clone https://github.com/USUARIO/Mushroom-Mix-Up.git
```

2. Abrir Unity Hub.
3. Seleccionar **Add Project**.
4. Buscar la carpeta del proyecto.
5. Abrir con la versión **Unity 6000.5.2f1**.
6. Ejecutar la escena principal.

# Organización del proyecto

**Assets:** Recursos principales del proyecto.
**Scripts:** Código fuente del videojuego.
**Scenes:** Escenas desarrolladas en Unity.
**Audio:** Música y efectos sonoros.
**Sprites:** Recursos gráficos e imágenes.
**Prefabs:** Objetos reutilizables.
**Materials:** Materiales utilizados por los modelos.

# Objetivo del proyecto

Desarrollar un videojuego funcional mediante Unity, implementando mecánicas de juego, recursos multimedia y herramientas de trabajo colaborativo que permitan cumplir con los objetivos establecidos para la actividad académica.


# Cómo jugar

1. Iniciar el videojuego.
2. Controlar al personaje mediante los controles establecidos.
3. Mantenerse sobre la plataforma del color mostrado.
4. Aplastar a los enemigos para pausar el movimiento de los mismos.
5. Alcanzar el objetivo propuesto evitando los obstáculos presentes durante la partida.
