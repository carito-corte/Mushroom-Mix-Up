# 🍄 Mushroom Mix-Up

## Description

**Mushroom Mix-Up** is a video game developed using **Unity** as part of the **Video Game Programming** course at the National Open and Distance University (UNAD). 
The project was carried out collaboratively using **GitHub** as a version control tool, which allowed for the integration of contributions from each team member.

The goal of the game is to provide an interactive experience in which the player must overcome the challenges presented by exploring the environment, interacting with various elements, and completing the objectives set during the game.

# Game Mechanics

1. Character movement.
2. Interaction with objects in the scene.
3. Scoring system.
4. Conditions for victory and defeat.
5. Background music and sound effects.

# Controls

| Action | Key |
|---------|-------|
| Move | W, A, S, D or Arrow Keys |
| Jump | Space |
| Smash | Space + Hover over NPC |

# Installation

1. Clone the repository.

```bash
git clone https://github.com/carito-corte/Mushroom-Mix-Up/tree/main
```

2. Open Unity Hub.
3. Select **Add Project**.
4. Locate the project folder.
5. Open it with **Unity 6000.5.2f1**.
6. Run the main scene.

# Project Organization

**Assets:** Main project resources.
**Scripts:** Video game source code.
**Scenes:** Scenes developed in Unity.
**Audio:** Music and sound effects.
**Sprites:** Graphic resources and images.
**Prefabs:** Reusable objects.
**Materials:** Materials used by the models.

# Project Objective

Develop a functional video game using Unity, implementing game mechanics, multimedia resources, and collaborative tools that enable the achievement of the objectives set for the academic activity.

# How to play

1. Start the video game.
2. Control the character using the designated controls.
3. Stay on the platform of the color shown.
4. Crush the enemies to stop them from moving.
5. Reach the objective while avoiding the obstacles that appear during the game.
